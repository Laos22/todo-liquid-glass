# Todo Liquid Glass

Современное React-приложение для управления задачами с эффектом "жидкого стекла" и анимациями.

## Функции

### Основные возможности
- ✅ Добавление новых задач
- ✅ Редактирование существующих задач
- ✅ Отметка выполненных задач
- ✅ Удаление задач
- ✅ Локальное хранилище (localStorage)

### Фильтрация задач
- **Все задачи** - показывает все задачи
- **Выполненные** - показывает только выполненные задачи (галочка)
- **Активные задачи** - показывает только незавершенные задачи

## Как использовать

1. Запустите проект: `npm run dev`
2. Откройте боковую панель (иконка стрелки)
3. Выберите нужную категорию задач
4. Список задач автоматически отфильтруется

## Технологии

- **Frontend**: React 19 + Vite 6
- **State Management**: Redux Toolkit + React Redux
- **UI/Styles**: Tailwind CSS + Liquid Glass эффекты
- **Animations**: Framer Motion
- **Icons**: Lucide React + React Icons

## Запуск проекта

```bash
npm install    # Установка зависимостей
npm run dev    # Запуск в dev-режиме
npm run build  # Сборка для production
npm run deploy # Деплой на GitHub Pages
```
